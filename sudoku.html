Sudoku Solver using Answer Set Programming

This project solves a 9×9 Sudoku puzzle using Answer Set Programming (ASP) with the DLV solver. Rather than writing a procedural algorithm, the puzzle is modeled using logical rules and constraints.

Logic & Constraints

% Each cell has exactly one number
{ place(R, C, N) : N = 1..9 } = 1 :- R = 1..9, C = 1..9.

% No duplicate numbers in any row
:- place(R, C1, N), place(R, C2, N), C1 != C2.

% No duplicate numbers in any column
:- place(R1, C, N), place(R2, C, N), R1 != R2.

% No duplicate numbers in any 3x3 subgrid
:- place(R1, C1, N), place(R2, C2, N),
   (R1+2)/3 = (R2+2)/3,
   (C1+2)/3 = (C2+2)/3,
   (R1, C1) != (R2, C2).

These constraints describe what a valid Sudoku solution looks like. The DLV solver computes an answer set that satisfies all of them simultaneously.
